## Prerequisites
- Node.js 18+ installed

## Installation Steps

### 1. Install Dependencies
```powershell
npm install
```

This will install all required packages:
- React 19
- Material UI
- React Router
- TypeScript
- Vite

### 2. Start Development Server
```powershell
npm run dev
```

The application will start at `http://localhost:5173`

### 3. Build for Production
```powershell
npm run build
```

Creates optimized production build in `dist/` folder

### 4. Preview Production Build
```powershell
npm run preview
```

5. Use **Previous/Next** buttons to navigate between topics
6. Follow the **Practical Exercises** for hands-on learning
